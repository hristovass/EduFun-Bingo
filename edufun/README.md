# About the App

**EDUfun** is an educational quiz application designed for elementary school students to learn and practice knowledge in a fun and interactive way.


## Overview

The app offers quizzes in **four categories**: History, Geography, Music, and Technology.  Each category contains **10 questions**, with **4 multiple-choice answers** per question. A timer runs for each question , adding a sense of challenge and urgency.  At the end of the quiz, users can see their **total score**, helping them track progress and motivating further learning.

## Installation and Setup

Follow the steps below to install and run the **EDUfun** application locally:

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/TP2304/EDUfun.git
2. **Install dependencies:**
   - cd EDUfun/client -> npm install
   - cd EDUfun/server -> npm install
3. **Run the server (backend):**


    In the main project directory (EDUfun), open a terminal and run:
      - node server.js
4. **Run the app (frontend):**

  
     Open a new terminal, navigate to the client folder, and run:
      - cd EDUfun/client -> npm start
  
5. **Open the app in your browser:**
   
        http://localhost:3000
